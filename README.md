# 📚 个人高级读书管理系统 V2.0
**Java 控制台实战项目 | 学生大作业/课程设计/个人作品集**
功能完善、数据持久化、支持分类、阅读进度、笔记管理、数据导出备份，远超基础版。

## ✨ 核心功能（15项）
### 书籍管理
- 添加、查看、修改、删除书籍
- 书籍分类管理（文学/科普/历史/教辅等）
- 全局关键词模糊搜索
- 阅读状态自动更新：未读 / 在读 / 已读
- 阅读进度（已读页数/总页数）实时记录
- 书籍评分功能

### 读书笔记管理
- 单本书籍多条笔记记录
- 笔记时间自动生成
- 笔记修改、删除
- 批量导出所有笔记为本地文本文件

### 数据与统计
- 本地文件持久化存储，重启数据不丢失
- 阅读数据统计（总书籍数、已读/在读数量、总笔记数）
- 数据一键备份
- 简易阅读计划提醒

## 🚀 运行方式
1. 导入项目到 IDEA / Eclipse
2. 直接运行 `Main.java`
3. 所有操作在控制台完成，数据自动保存到 `books.data`

## 💡 项目亮点
1. 实现**序列化持久化**，关闭程序数据不丢失
2. 完整的增删改查、搜索、分类、统计功能
3. 输入友好、控制台菜单清晰
4. 代码结构规范，可直接作为Java课程设计、GitHub作品集
